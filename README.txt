Hello fellow sw friend, to exec this you can:

--------------------------------------------------------------------------------
                                Linux Users:
  1) Close ALL chrome tabs
     Make sure the CG_PROJ_4 folder is in your desktop
     Edit 'USER' variable in the shellscript CG_PROJ_4.sh

        $~ chmod +x CG_PROJ_4.sh
        $~ ./CG_PROJ_4.sh\

  2) Run chrome with the flag --allow-file-access-from-files:
     $~ google-chrome --allow-file-access-from-files
     After chrome opens you can simply open main.html as you always do :)
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
                                  Mac Users:
  1) Just the same as linux users but please confirm that the path in the
     shellscript is the same.
